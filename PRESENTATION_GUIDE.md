# CareerTrust Presentation Guide

## 30-Second Pitch
CareerTrust is an AI-powered career platform that helps students discover legitimate opportunities faster. It aggregates live job sources, ranks opportunities against user skills, and adds a trust layer so users can review matches with more confidence.

## What To Show In A Demo
1. Open the home page and explain the product value in one sentence.
2. Go to Profile and add a few skills plus a resume.
3. Open Recommendations to show match scores, missing skills, and trust scores.
4. Open Job Integration to fetch live jobs from RemoteOK, Dev.to, or all sources.
5. Import jobs and show the Opportunities page.
6. End on the Dashboard to show profile completion, saved jobs, and activity.

## Talking Points For Industrial Reviewers
- Real job aggregation instead of static mock data.
- AI-assisted ranking using skill matching and TF-IDF scoring.
- Trust scoring and scam-awareness messaging to support safer decisions.
- Clean, guided workflow for students and early-career candidates.
- Clear upgrade path for premium APIs, stronger detection models, and production deployment.
